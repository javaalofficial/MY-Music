Manajer Plugin

Upload file .js untuk menambahkan fitur. Plugin tersimpan permanen dan aktif otomatis setiap kali app dibuka.

Format plugin:
window.MusicPlugins.register({
  id: 'nama-plugin',
  name: 'Nama Plugin',
  init: function(api) {
    // api.getAudio(), api.getState()
    // api.addPluginPanel(id, judul, fn)
    // api.showNotification(msg)
  }
});

✓ Contoh plugin mini (copy ke file .js):
window.MusicPlugins.register({id:'demo',name:'Demo',init:function(api){var s=api.getState();api.showNotification('Lagu: '+s.playlist.length);api.addPluginPanel('demo-p','Status',function(c){c.innerHTML='<b>'+s.playlist.length+'</b> lagu'});}});
✓ Plugin VALID: Skrip JavaScript biasa dengan window.MusicPlugins.register()
✗ Plugin TIDAK VALID: File dengan import/export, top-level await, atau tanpa .register()
